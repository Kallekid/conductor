# Changelog

All notable changes to **Conductor SDLC** will be documented in this file.

## [1.0.0] - 2026-03-08

### Added
- **Initial Transformation**: Forked from original Conductor and rebranded to **Conductor SDLC**.
- **SDLC Master (`/conductor:newProject`)**: New comprehensive initialization engine with multi-persona interviews (PO, Dev, QA).
- **Architectural Enforcement**: Built-in support for Repository Pattern and Dependency Cruiser rules.
- **Advanced Workflow**: Integrated Anti-Mocking philosophy, Scouting-first mentality, and PR-based track lifecycle.
- **Skill Creation (`/conductor:createSkill`)**: New command to scaffold and register custom AI skills.
- **Tool-Tagging**: Mandatory skill and MCP tagging for all tasks in implementation plans.
- **Resource Budgets**: Capacity to define and verify CPU/RAM/Latency targets in specifications.
